# Network Traffic Analysis Using Wireshark

## Description
This project focuses on capturing and analyzing network traffic using Wireshark to understand how data flows within a network. It helps in identifying protocols, monitoring packets, and detecting abnormal or suspicious activities.

## Objective
- To capture live network packets
- To analyze common network protocols
- To understand network behavior and traffic patterns
- To identify potential security issues

## Tools Used
- Wireshark
- Local Area Network (LAN)
- System with active network connection

## Methodology
- Captured live network traffic using Wireshark
- Applied filters to analyze specific protocols such as TCP, UDP, HTTP, and DNS
- Observed packet details including source, destination, and packet size
- Identified normal and abnormal traffic patterns

## Outcome
- Gained practical understanding of network traffic flow
- Learned protocol-level communication
- Improved skills in network monitoring and troubleshooting
- Developed awareness of network security analysis

## Applications
- Network monitoring
- Troubleshooting network issues
- Cyber security analysis
- Detecting suspicious or malicious traffic
